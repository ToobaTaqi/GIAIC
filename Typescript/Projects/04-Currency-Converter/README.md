## Task Given
The TypeScript console app is used to convert currencies: the users enter a certain amount of money in one currency and set the currency they want to check the monetary value of.

While developing the app, the beginners can master variables, algorithms, loops, if statements, and other TypeScript concepts.

## Currency Converter
This is a simple Currency converter application built with TypeScript, `inquirer`, and `chalk`. It allows a user to convert an amount into its desired currency from another one.

## Getting Started

follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the project directory in your terminal.
3. Install dependencies by running `npm install`.
4. Start the ATM by running `tsc && node index.js`.
5. Follow the prompts to enjoy the Functionalities.


## Dependencies

- Node.js
- Inquirer
- Chalk
